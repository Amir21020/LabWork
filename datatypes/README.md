# Типы данных

## Описание

Две задачи на работу с типами данных.

### Task1 — Калькулятор сложных процентов

Расчёт накоплений с капитализацией процентов (год за годом).

Метод: `Calculator.CalculateCompoundInterest(decimal initialDeposit, int years, decimal interestRate)`

### Task2 — Генерация ромба из символов X

Написать функцию, которая выводит на экран ромб (диамант) из символов X, где:

- Длина каждой диагонали равна N (положительное нечётное целое число)
- Центр ромба должен быть пустым (пробел вместо X)
- Значение N передаётся в функцию в качестве аргумента

Метод: `DiamondPrinter.GenerateDiamondFromSymbolX(int N)`

## Файлы

- `Task1/Calculator.cs` — калькулятор процентов
- `Task2/DiamondPrinter.cs` — ромб из X
