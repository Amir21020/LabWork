# ООП. Геометрия — прямоугольники

## Описание

Класс `Rectangle`, задающий прямоугольник с координатами левого верхнего угла (`X`, `Y`), шириной (`Width`) и высотой (`Height`).

## Свойства

- `X`, `Y` — координаты левого верхнего угла (отрицательные значения запрещены)
- `Width`, `Height` — размеры (должны быть строго больше нуля)
- `Area` — площадь (вычисляемое, только чтение)
- `Perimeter` — периметр (вычисляемое, только чтение)

- `Rectange.cs` — реализация класса (опечатка в имени файла)
