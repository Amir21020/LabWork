# Практика

## Структура

- `datatypes/` — задание с типами данных (калькулятор сложных процентов, генерация ромба из символов X)
- `oop/` — задание с ООП (класс Rectangle с инкапсуляцией, валидацией, вычисляемыми свойствами Area/Perimeter)
- `collection/` — задание с коллекцией (реализация «Умный стек» — SmartStack\<T\>)
- `db/` — задание с БД (MSSQL, университет)
- `http/` — задание по HTTP (работа с API Центробанка и Dummy REST API)
- `orm/` — задание по ORM (ADO.NET vs Entity Framework 6, CRUD Tasks)
- `web/` — Финальное задание веб-приложение **TimeTracker** (ASP.NET Core Web API + Vue 3 + PostgreSQL)
