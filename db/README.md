# База данных "Университет"

## Структура БД

- **departments** — кафедры (название, бюджет, дата основания)
- **groups** — учебные группы (название, средний балл, привязка к кафедре)
- **students** — студенты (имя, фамилия, стипендия, телефон, привязка к группе)
- **teachers** — преподаватели (имя, фамилия, зарплата, дата найма)
- **lessons** — занятия (предмет, время, флаг онлайн, привязка к преподавателю и группе)

## Файлы

- `init_db.sql` — создание базы `university` и всех таблиц
- `queries.sql` — примеры запросов: выборка, группировка, обновление, удаление, объединения таблиц

## Запуск

Выполните `init_db.sql` в СУБД (MSSQL), затем `queries.sql` для проверки запросов.
