# Коллекция (SmartStack)

## Описание

Реализация обобщённого стека `SmartStack<T>` с динамическим расширением внутреннего массива.

## Методы

- `Push` — добавление элемента на вершину
- `Pop` — удаление и возврат элемента с вершины
- `Peek` — просмотр вершины без удаления
- `PushRange` — добавление набора элементов
- `Contains` — проверка наличия элемента
- Индексатор `this[int index]` — доступ по индексу от вершины (0 — вершина)
- Реализует `IEnumerable<T>` (перебор от вершины к низу)

## Конструкторы

- `SmartStack()` — начальная ёмкость 4
- `SmartStack(int capacity)` — с указанной ёмкостью
- `SmartStack(IEnumerable<T> collection)` — заполнение из коллекции
